## Project OverView
This is an app for people to look for their favorite recipes.
Users can look for a recipe on the seach bar and users can upload new recipe and bookmark recipes.

## Features

User look for a recipe
Add new recipe
Bookmark recipe
Unbookmark recipes
Select favorite recipes.
